# GEMINI IA API
  Este repositorio contiene un ejemplo de una API que utiliza **GeminiIA** para el procesamiento de lenguaje natural. La API se probó y configuró utilizando **Postman**
## Requisitos Previos
  - Cuenta en **GeminiIA**.
  - **Postman** instalado para probar la API.
## Configuración
